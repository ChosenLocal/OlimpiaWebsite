# Service Page Template — Content Outline

1) H1 with location modifier (e.g., “Crime Scene Cleanup in Portland Metro”)
2) Safety-forward intro (answer-first, 40–60 words)
3) Process (steps 1–5) — optional HowTo schema
4) Timelines (typical durations; no promises)
5) Insurance handling
6) Local proof (“recent work near {City}”) — redact details
7) FAQ (5–8 Qs) — FAQPage schema
8) CTA bands every ~700 words (Call/Chat/Estimate)
