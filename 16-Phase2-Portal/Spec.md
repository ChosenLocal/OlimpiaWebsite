# Adjuster / Partner Portal (Phase 2)

## Roles
- Adjuster (external), Technician (internal), Admin.

## Features
- Secure doc exchange (estimates, invoices, photos), watermarking.
- Chain of custody logs; activity audit trail.
- Case notes with mention notifications.
- SSO (Magic links) + short-lived access.

## Compliance
- PII minimization, encrypted storage, signed URLs with TTL.
